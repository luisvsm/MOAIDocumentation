Android NDK: ERROR:luajit/Android.mk:luajit: LOCAL_SRC_FILES points to a missing file


The problem is in the lua jit building, try changing the following to false.

Code: Select all
  1.  

  2. use_fmod="false"

  3. use_untz="true"

  4. use_luajit="true"

  5.
