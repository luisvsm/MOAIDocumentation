- play the app, everything is fine, 60 FPS, smooth

but now the frame rate is more like 30 FPS

Seems to be related to the back button function - like some thread isn't being cancelled, polling going on?

We've run out of ideas, anyone seen anything like this?

Re: Weird kindle bug with startActivity

by scottwb » Sun Dec 08, 2013 12:20 pm

I think we've solved it ...

It seems that we need in the manifest

Code: Select all
  1.  

  2. android:hardwareAccelerated="true"

  3.  



despite the kindle fire 7" 2nd gen being api level 15:

https://developer.amazon.com/sdk/fire/specifications.html

Android docs say it's hardware acceleration is auto enabled for >= api level 14:

http://developer.android.com/guide/topics/graphics/hardware-accel.html
