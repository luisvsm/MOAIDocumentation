Nice OOP structure and pretty easy to customize and extend.

If you decide to go with MOAI, then be ready for possible problems:
  • Hanappe is designed for games (not only mobile), so you'll find that some common mobile widgets are missing. There is no table/list view.
  • Input fields can be buggy. This is due mobile API's are not designed to display input fields in OpenGL app without hacks and workarounds.
  • You may find that some features are missing in Facebook bindings and you'll need to dive into C++ part to expose missing functionality.
