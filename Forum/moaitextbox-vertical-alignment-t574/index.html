Re: MOAITextBox vertical alignment

by dana » Mon Mar 19, 2012 7:07 pm

Hi silverson,

In 1.0, the implementation of the TextBox class will be changed substantially. Among the changes is an option for a vertical alignment as you requested. You will set the vertical alignment by passing in the alignment as the second parameter to setAlignment. I can go into the other TextBox changes for 1.0 if you are interested in them, also.

To get the size of the TextBox, you should be able to call textbox:getRect () and it'll return the dimensions of the TextBox. Is this what you were wanting, or am I misunderstanding?

by dana » Mon Mar 19, 2012 11:14 pm

The issue with the text aliasing when center aligned will also be fixed in 1.0.

You're right in that getRect returns the size of the text box, not necessarily the size of the text that it contains. As you're already doing, the getStringBounds function should return the dimensions of the text.

Although I am wondering if you're only looking for the height in order to do the centering yourself? If this is the case, it sounds like come 1.0 when vertical alignment and the aliasing from center alignment is fixed, you may not even need this.

The plan is to launch 1.0 on Friday.

Let me know how it goes.
