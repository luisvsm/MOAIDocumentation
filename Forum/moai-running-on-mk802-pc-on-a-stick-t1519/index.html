MOAI running on MK802 PC-on-a-stick ..

by ibisum » Sat Dec 29, 2012 4:13 am

For grins and giggles I got an MK802 PC-on-a-stick recently, and wired it up to my lapdock .. installed lubuntu-mk802 (Ubuntu with lxde wm) .. and just spent the last hour building shadowmints' fork of MOAI for Linux.

And: it works! So this means that you can plug in a $50 device into your home TV and run MOAI on it, nice and smooth!

Just wanted to share that detail, in case anyone is looking for a new toy to play with .. and I have to say that having MOAI onboard such a small, light, cheap device just feels really, really good ..

EDIT: its this thing: https://www.miniand.com/products/MK802% ... 0Mini%20PC

Also, it runs Android out of the box .. and all my existing MOAI apps run smoothly on it, too, in Android mode ..

Re: MOAI running on MK802 PC-on-a-stick ..

by ibisum » Sat Dec 29, 2012 3:01 pm

I've got a few of these things on my desk at the moment, the MK802 was just the first one I picked up to use, no reason. I've got a Motorola Lapdock wired up for them all, its a terrific setup to be honest and if it weren't for the jury-rigging I'd probably consider using this as a netbook/laptop replacement in the near future. Maybe if I get some time on a 3D printer to create a suitable housing.. but if you're just going to plug it into your TV for casual use, you can probably just make do with the DroidMote server that they come with, which lets you control the thing from another Android phone .. for my needs, I won't be running Android on it much, Ubuntu instead, and this means I can get away with using Synergy too, which is already used in my setup.

Let me know if you get something like this - it sure is nice to see MOAI running on such inexpensive hardware. I think there could be a viable business with taking these things, installing custom software on them, and re-selling them .. especially when, in bulk, you can get MK803/MK808 hardware at $15/each. !!!

Re: MOAI running on MK802 PC-on-a-stick ..

by ibisum » Sun Dec 30, 2012 4:55 pm

I made a custom cable. The lapdock has USB out and HDMI out, so its really just a mouse/keyboard/HDMI-monitor in a nice laptop-shaped package - albeit, the connectors are the wrong sex. So I got a few adapters and built my own cable to go from the Lapdock outputs to the MK802 inputs. Its all quite well documented here:

http://www.robpol86.com/index.php/Atrix ... Other_Uses

As well as all over the web:

http://www.youtube.com/watch?v=VfdKq33WgHw


I have to say, its a pretty damn nice machine once you get it working, and if but for the cable mess, I'd be using it constantly. Luckily, I have access to a laser cutter and 3d printer, so hopefully some time this week I'll build a housing for the rig that makes it a lot more robust.
