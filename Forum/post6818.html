Re: Untz sound problem on iPad

by flyit » Wed May 23, 2012 12:30 am

I found the cause.

Because Moai sdk 1.1 changed some codes in MoaiView.mm from sdk 1.0.

1.0:
AKUUntzInit ();

1.1:
#ifdef USE_UNTZ
AKUUntzInit ();
#endif

So, we need to add -DUSE_UNTZ in Other C Flags at xcode project settings. Then we can play with sound.
