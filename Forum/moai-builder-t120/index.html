Moai Builder

by dlannan68 » Fri Sep 02, 2011 9:14 pm

Ok, I have finally managed to get some external dll's working within an IUP Lua framework, all launched within a Moai context.

Heres the initial setup (wont be staying like this - will end up more Web2 like in overall looks).

The good things:

- All uses standard Lua libraries (portable to other platforms)

- Moai execution will be available _within_ the main view.

- Multiple Moai executions will likely be possible (for multiple projects or client/server testing)

- All Moai objects should be available as components (if you are familiar with Unity3D - this is similar).

- Since its _entirely_ written in Lua, easily extensible, and modifable to suit your own needs :)

The bad...

- Some very odd problems with IUP and its module loading (need to ensure this is portable)

- Not sure if I can integrate with cloud (Im hoping so - so you can preview, upload, run).

- Time... I need to quit work or something.. :)

More to come..



Open to any ideas, tips, requests.. exciting stuff this is :)

Cheers,

Dave.
