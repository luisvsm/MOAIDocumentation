it expects an environment variable called ANDROID_NDK to be defined. You can either add a line to the top of the script like
Code: Select all
  1.  

  2. ANDROID_NDK=<path to ndk here>

  3.  



or
Code: Select all
  1.  

  2. export ANDROID_NDK=<path to ndk here>

  3.  


from the command line before running the script

by mattpace
One thing that needs to be done is tell OSX that when running sudo, don't clear the ANDROID_NDK environment variable.

At the command line, run
Code: Select all
  1. sudo visudo

. Add the line
Code: Select all
  1. Defaults    env_keep += "ANDROID_NDK"

below the other similar lines. It's vi so if you're not familiar, scroll down to the line, hit 'i', add the line, then [Esc] ':wq' [Enter].
