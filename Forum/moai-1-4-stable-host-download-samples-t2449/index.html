by dorotik » Wed Jan 01, 2014 5:49 pm

Hi,

do you want to start playing around with Moai but cannot find it anywhere? Or you got it but it doesn't work? Well there's a link to a package which will sure help you to get things started:

Moai-Lightweight - Download Moai Last Stable Release + Samples - Version 1.4p0:
https://mega.co.nz/#!SE8x0CxK!Bu-KnVh3St7TtsuHKSJbCi-AI3ELEShwyfIcm3-BFKM

Lightweight means that there are no source codes, but you'll of course get a fully fledged version of Moai. To download sources, check Moai GitHub at https://github.com/moai/moai-dev or download their last stable version using this link: https://github.com/moai/moai-dev/archive/Version-1.4p0.zip

In the Moai-Lightweight package, look into samples/Getting samples to run on Windows.txt file. It will show you how to get Moai working on your machine (it's easy).

All samples were tested on Windows 8.1, but I hope they will work on Windows 7 too. Samples which didn't work properly were placed into samples/win_incompatible_or_broken folder. Anyway, check out their .lua sources too, there's plenty of knowledge to learn from them.

BTW: OS X - Last stable Moai host is there also.

Welcome to Moai!
