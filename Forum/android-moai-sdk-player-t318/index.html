Android Moai SDK Player


Allows the user to point to a url containing a lua file which will be downloaded and executed within the Moai Environment.



Useful for trying Moai LUA code on an android device.

Just publish your lua file on the internet and point the applications url path to it. The application will download the lua file and execute it.



Using a bootstrap technique the file can then subsequently pull down further lua files or assets.

(see https://github.com/latestpost/Moai-Examples/tree/master/bootstrap)
