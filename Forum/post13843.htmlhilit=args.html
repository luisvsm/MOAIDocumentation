by halfnelson » Fri Mar 28, 2014 9:36 pm

Hi!, The cmake system is pretty new and still has some rough edges. It has not been optimised for use with the CMake GUI. In fact I have not tested it with the CMake GUI. It is best to stick to the scripts in the bin folder.

You can use the create-projects-ios.sh file to generate xcode projects and build from inside the IDE if that makes it easier. The CMake program has no concept of iOS so all options for iOS will show up in the gui as OSX options or Xcode options.
