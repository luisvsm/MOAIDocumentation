Saving/Restoring State

Posted: Fri Dec 16, 2011 3:46 pm
by patrick
 MOAISerializer works pretty well. It will handle cycles of Lua tables, but not functions or closures. It also can serialize/deserialize Moai framework objects, though we've only written actual serialization code for a few of the objects (MOAIGrid and MOAITransform, I believe).

Supporting functions/closures and writing handlers for the rest of the framework is on the list, but has been more about as-needed for our own projects. If you need more of the framework serialized LMK.

Of course you can also use Lua pickle or pluto. They are fine for game state, and pluto handles functions and closures (I think) though neither of these will serialize Moai objects.
