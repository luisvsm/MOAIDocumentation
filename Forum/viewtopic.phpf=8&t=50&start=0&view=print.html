Terribly sorry about that; I was actually running some development code that included the inclusion of the Hello World app. The "Create Application" button is the same as the "New Service" button, just different wording. To get your Hello World app running, you just need to create a lua file named "main.lua" and include the Hello World code that was listed in the middle of the tutorial:
[lua]function main(web, req)
  web:page("hello world!", 200, 'OK')
end[/lua]
Zip that file up by itself and upload it; your app will now be added.
