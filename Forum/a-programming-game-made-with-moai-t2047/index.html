A programming game made with moai

by halfnelson » Sun Sep 08, 2013 9:41 pm

I have been plugging away on a programming game as a way to learn lua and moai, and to scratch a personal itch . I have a work in progress implementation on the web at http://moaijs.moaiforge.com/progrocket/index.html (requires WebGL).

Would love some feedback as to which parts need more tutorial assistance, and if you can think of any other instructions/concepts that would be fun to add. I am already considering an "analog" physics based version for extra possibilities.

Note: It saves your progress in the browser local storage so you can go back to it later.
