Re: Circular Game Environment.

by Vavius » Tue Apr 14, 2015 4:17 am

naturally wrote:Use 2 duplicate Props per piece.
Even 4 to handle corner situations.

Here is the MOAI pro way:

Render whole layer 8 more times with offsets. We can create layers and viewports and use partition from the main layer. You don't need 4 props for each object and checks when to enable/disable them. Since MOAI performs clipping for us, the overall performance hit will be minimal. Positions of all objects can be wrapped with modulo operation but they will still move smoothly.

Code: Select all
  1. offsets = {

  2.     { 2,  0},

  3.     {-2,  0},

  4.     { 0,  2},

  5.     { 0, -2},

  6.     { 2,  2},

  7.     {-2,  2},

  8.     {-2, -2},

  9.     { 2, -2},

  10. }

  11.  

  12. -- layer here is your main game layer which you want to wrap around

  13. partition = layer:getPartition ()

  14.  

  15. for i = 1, 8 do

  16.     local viewport = MOAIViewport.new()

  17.    

  18.     -- viewport settings should be the same as in your main viewport

  19.     viewport:setSize ( 320, 480 )

  20.     viewport:setScale ( 320, -480 )

  21.     viewport:setOffset ( unpack ( offsets [ i ] ))

  22.  

  23.     local layer = MOAILayer2D.new ()

  24.     layer:setViewport ( viewport )

  25.     MOAISim.pushRenderPass ( layer )

  26.  

  27.     layer:setPartition ( partition )

  28. end
