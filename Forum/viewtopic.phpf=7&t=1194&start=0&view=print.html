ZeroBraneStudio's integration with Moai is now with autocomp

Posted: Thu Sep 20, 2012 5:36 pm
by paulclinger
I just added Moai autocomplete to ZeroBrane Studio: http://studio.zerobrane.com/. You would need to get the latest version from the git repository; this will be packaged in the next release.

It should work when you type "MOAI|", but it also works when you type (you will get the list of methods for MOAIProp2D after prop:):

local prop = MOAIProp2D.new()
prop:|

It does a bit of type guessing, but it's really simple and not fool proof, even though it should work in many situations just fine.

Paul.
