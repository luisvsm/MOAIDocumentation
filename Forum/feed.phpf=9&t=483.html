You have run make-host.sh and passed it a project name in Java form right? Something like:

cd /cygdrive/c/moai/ant/
./make-host.sh -p yourappname.yourdomainname.com


Leave the terminal open. After this, confirm that under the ant directory, a folder named untitled-host has been created. ( ls untitled-host If that returns cannot access untitled-host, it wasnt created ).

Assuming it does exist

At this point you need to change in to the untitled-host folder as described in the guide and edit settings-local.sh. You can use notepad, or whatever to edit the values. Do so and save. Now back at the terminal window

cd untitled-host
./run-host.sh

If this fails, paste the exact results from the terminal window here.
