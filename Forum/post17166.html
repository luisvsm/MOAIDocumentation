Re: Pendulum Clock (Looking for Beta Testers)

by jglascoe » Wed Apr 22, 2015 6:16 am

It only runs when the MoaiView it's on thinks it's visible. So if you open a chrome window and do some browsing, Moai will pause and the clock will basically sleep. When you close the browser and switch back to a desktop view, Moai will un-pause and the clock will realize its displayed time is off by more than a couple seconds and it will (nearly) instantly set all the wheel and pinion rotations to what they should be.

For free versus paid: I'm sure the best way to monetize is in-app purchases, but I don't want to spend another hundred hours coding this functionality. Supporting two separate releases is so much easier. That said, for free I'm currently thinking to include both black and white backgrounds, all the brass colored clock elements, and support options: Not Visible, and Brass for the clock elements. It still looks really nice and you have the ability to remove/hide some elements to better see how all the gears interact with each other.

The paid version would include many more asset files and support options: Not Visible, Brass, Steel, Black Outline, White Outline.
