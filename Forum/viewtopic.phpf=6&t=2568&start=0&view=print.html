Hi Stun,

Thank you very much!

For technical challenges I can say that sound was a big issue. For android especially...

On android:
First I had serious clipping issues with Untz, I worked through that by capping the volume and setup a sound pool. That worked but I was still getting crashes and at first I did not know they were related to the sound. The way the hosts are setup it would have taken quite a while to get gbd working to get error output, and adb only shows the lua-level errors. So, I went over to iOs and tinkered and finally found that the crash was from within Untz (well, I had a good feeling anyways).
So, I forked my moai builds for my hosts and gutted the untz sound object to simply hold the info on the sound/music that I wanted to play. I then wired up my android host with JNI and used the android sound pool and media player.

That meant I got great sound on android and I used untz on ios and NO code changes on the lua level (w00t)!


I also had memory crashes on ios that drove me nearly insane. My game was crashing and I was getting crash reports that DID NOT show my game being jettisioned. totally wtf moment right there. So I changed up my lua code to handle texture swapping better (I was loading a LOT at first and running with it). That didn't solve it and memory usage was really low. I finally reboot my ipad and bingo, worked without any issues. There is something very fishey going on inside iOS, not sure what though.


What else...I didn't use rapa nui or hanape...I rolled my own stuff for that early on; little MVC with front controller. I didn't do anything special for texture management...I would if I did that again.


Setup some arraylists and linked lists early on with function that return object, that worked really nice.


One funny thing...the drawing functions to very funny things sometimes, you have to be careful to give them proper values. And my sound still does some funny things on some android devices.


That's about all I can think of for now Stun.


I don't have a blog sorry, I probably should have set one up on the way....my bad.

Thank you!
Jonas
