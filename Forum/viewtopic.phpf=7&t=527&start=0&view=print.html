HowTo Debuging Lua Code quick and dirty

Posted: Mon Feb 20, 2012 4:20 pm
by Vad1mo
You are reading this because you want to find out how you can quick and dirty debug your lua code.

1. get eclipse Lua Development Tools http://www.eclipse.org/koneki/ldt/
2. Read this section http://wiki.eclipse.org/Koneki/LDT/User ... mote_Debug and copy the debugclient files next to main.lua
3. Enable lua sockets in moai host or take this bin files (eg win32) http://files.luaforge.net/releases/luas ... 32-vc8.zip
4. paste require("debugger")("127.0.0.1", nil, nil) somewhere in your main.lua code.
5. set a breakpoint in eclipse
6. Start remote debugging
7. Start the game.

wait until your program stops at the breakpoint

enjoy..
