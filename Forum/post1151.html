MOAIGrid confusion

by patrick » Wed Dec 07, 2011 5:04 pm

Glad you are enjoying Moai! 

When you're initializing your tile deck you can specify an optional offset and tile size. So something like:

tiledeck:setSize ( 8, 8, 32, 32, 1, 1, 30, 30 )

would specify a tile size of 30x30 offset 1, 1 from the corner of a 32x32 cell in a grid 8 cells wide and 8 cells tall. Note that the grid does not have to fill the texture. In our games we've been using mostly 64x64 tiles with a one pixel 'bleed' around them. So the artists leave some room at the edge of the texture to account for the bleed. The cell size is 66x66 with the 1, 1 offset to get a 64x64 tile.

Let me know if that makes sense.

(Actually, I think that the cell, offset and tile dimensions need to be in UV coordinates... so divide the parameters above by your texture dimensions.)
