If you are using a new Android Studio build which is based on Gradle this setting is found in the build.gradle file. I had problems with this as well because my manifest changes were not updated in play store. If you are using gradle/studio just remove <uses-sdk> from the manifest because its just overwritten with gradle settings anyway.

build.gradle
Code: Select all
  1. apply plugin: 'com.android.application'

  2.  

  3. android {

  4.     compileSdkVersion 20

  5.     buildToolsVersion "20.0.0"

  6.  

  7.     defaultConfig {

  8.         applicationId "se.skyturns"

  9.         minSdkVersion 11

  10.         targetSdkVersion 19

  11. ...

  12.
