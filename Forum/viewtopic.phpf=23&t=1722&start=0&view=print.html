As far as I can tell it is currently not possible to pan sound effects in moai, isn't it? Positioning it in 2D/3D space with a listening camera would be the same as panning the sound. Please note I do not want to 'position' the cursor on the sound effects timeline. I want to set the pan (left/right channel balance) of a sound.

  • Sound effects can not be parented to a prop which would be the most obvious way to set '2D/3D position of sounds'. like this...
    Code: Select all
    1. mysound:setParent ( myprop )

    2. -- or

    3. mysound:setAttrLink ( MOAIProp2D.INHERIT_TRANSFORM, myprop, MOAIProp2D.TRANSFORM_TRAIT )

  • Untz c-code does not provide such a function at all.
    https://github.com/moai/moai-dev/blob/m ... ntzSound.h
  • Fmod should be able to pan a sound (FMOD_Channel_SetPan) but it is not forwarded by moai wrapper
    https://github.com/moai/moai-dev/blob/master/src/moaiext-fmod-ex/MOAIFmodExChannel.h

Is panning not included to keep untz and fmod interface similar?
Adding panning to fmod ext should trivial as far as I can tell. Is it planed for a future release? Can anyone of moai staff comment on this?
