Windows / Linux and OSX are based on Glut, which is not recommended for production.
It should be easy to get a new host using SDL or just OpenGL for these platforms.

Android, OSX and Chrome's Native Client are ok for production.

For windows you can check Akuma, post3951.html?hilit=sdl#p3951

I admit I haven't checked the V1.4, but in the previous releases iOS was not fully ready for production out-of-the-box as it didn't support Landscape orientations very well. Somewhere else on this forum I've posted my "hacks" to support reliably landscape orientation on iOS.

Additionally, I think you shouldn't remember to package lua files in some format that will not be visible if the user opens the package. The way it is by default, lua files will be visible to end user which may not be a desired condition.
