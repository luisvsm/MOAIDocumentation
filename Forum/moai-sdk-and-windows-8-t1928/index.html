Re: Moai-sdk and Windows 8

by BacioiuC » Sun Jul 14, 2013 2:42 pm

Before you do any changes to your build make sure you have the latest graphic card drivers. I had the same issue while developing my game on Windows 8 and, just like you, ran into the same issue. The command prompt would appear, outputting everything but the main window would not render and it would crash with an error. I solved it by updating my drivers to the latest version. The issue came up on both mine and my employers computer and it had the same fix.

So bottom line, do a update of the drivers. The default ones supplied by windows 8 have problems with OpenGL bindings.
