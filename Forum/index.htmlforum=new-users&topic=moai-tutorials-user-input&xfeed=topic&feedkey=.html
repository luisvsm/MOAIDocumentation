I need halp, what is a cluster, and i looked at your hello tutorial, your cluster is public, mine is shared. Can you explain?

Sorry for the confusion.  The term "public" is a typo in the tutorial documentation.  It should be "shared."

A Moai Cloud cluster is a group of redundant, load-balanced application servers that are responsible for hosting your Moai Cloud applications.  The default cluster is "shared", meaning that your application is running on a cluster that is shared with other Moai Cloud customers.  In most cases, the shared cluster will be adequate for supporting your application.  

However, if you find that your Moai Cloud application is receiving large volumes of traffic or if you need more functionality than is provided by the default Lua sandbox, we can create a private, dedicated cluster for you that will be used solely for your applications.  If you're interested in setting up a private cluster, please let us know and we'll work with you to get one set up.
