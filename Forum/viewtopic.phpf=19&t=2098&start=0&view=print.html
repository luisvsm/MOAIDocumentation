I tried to complile moai with VS2010 and moai.exe crached too. Then I try to use bat file in moai-dev\bin\build-vs2010.bat
and I get error - solution is not make. Then I read errors and edit next files
1) bin/build-vs2010.bat
---> vs2010 replace with ..\vs2010
Code: Select all
  1. @@ -1,7 +1,7 @@

  2.  call "C:\Program Files\Microsoft Visual Studio 10.0\VC\vcvarsall.bat"

  3. -cd vs2010

  4. +cd ..\vs2010

  5.  

  6.  msbuild moai.sln /property:Configuration=Debug-Lua-5.1

  7.  if ERRORLEVEL 1 exit

  8.  

  9.  msbuild moai.sln /property:Configuration=Release-Lua-5.1


2)Patch VS2010
edit %ProgramsFile%\MSBuild\Microsoft.cpp\v4.0\Microsoft.CPPBuild.Targets
patch
Code: Select all
  1. --  <Target Name="GetResolvedLinkLibs" Returns="@(LibFullPath)" DependsOnTargets="$(CommonBuildOnlyTargets)">

  2. ++  <Target Name="GetResolvedLinkLibs" Returns="@(LibFullPath)" DependsOnTargets="$(CommonBuildOnlyTargets);ResolvedLinkLib">


3) Install FMOD Ex Designer (http://www.fmod.org/download/)
4) Install latest Direct X SDK (http://www.microsoft.com/en-us/download ... px?id=6812)
5) Edit moai-dev\vs2010\moai-lib-fmod-designer\moai-lib-fmod-designer.vcxproj and moai-dev\vs2010\moai-lib-fmod-ex\moai-lib-fmod-ex.vcxproj
add in all <AdditionalIncludeDirectories> ==> C:\Program Files\FMOD SoundSystem\FMOD Programmers API Win32\api\inc;C:\Program Files\FMOD SoundSystem\FMOD Programmers API Windows\fmoddesignerapi\api\inc
6) Edit moai-dev\vs2010\moai-exe-fmod\moai-fmod.vcxproj
add in all <AdditionalLibraryDirectories> ==> C:\Program Files\FMOD SoundSystem\FMOD Programmers API Win32\api\lib; C:\Program Files\FMOD SoundSystem\FMOD Programmers API Windows\fmoddesignerapi\api\lib

After that I ran bin/build-vs2010.bat and it compile and build OK. Then I copied moai-dev\vs2010\bin\win32\Release-Lua-5.2\*.* to c:\moai-sdk. After that all samples works in zerobrane fine
