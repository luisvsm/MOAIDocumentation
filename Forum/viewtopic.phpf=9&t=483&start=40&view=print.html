Posted: Sat Aug 04, 2012 1:29 am
by sgeos
Did you get the latest source from GitHub? This is a copy of what I posted on page 2:

Code: Select all
  1. # (install Android SDK and NDK)

  2. # (add SDK and NDK to path; export PATH=$PATH:/(SDK_PATH)/tools:/(SDK_PATH)/platform-tools/:/(NDK_PATH)/)

  3. mkdir ~/moai-source

  4. cd ~/moai-source

  5. git clone https:/ /github.com/moai/moai-dev

  6. cd moai-dev/ant/

  7. ./make-host.sh -p com.mycompany.app.appname

  8. cd untitled-host

  9. vi settings-local.sh

  10. # (configure android_sdk_root and src_dirs)

  11. ./run-host.sh


NOTE: That should be https://github... without the space on line 5.

by sgeos
Don't use the official build on Windows. If you want to do Android development, it is just asking for trouble. It is so much easier to build from source... and preferably not on Windows. =) (Others like Ubuntu. I use Mac OSX.)

EDIT: There is nothing wrong with developing you app on Windows using the glut host if that is your preferred platform. It is, however, a good idea to build from source on another OS when you are gearing up to package your APK.
