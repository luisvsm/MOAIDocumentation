Seems like your memory is excellent! Those were just the steps I needed.

For future reference, here are the same steps after I looked everything up:

host-templates\html\ <-- This is the template folder
host-templates\html\src\ <-- Copy your assets into this folder. You'll have to create it first.
host-templates\html\ <-- Run buildrom.bat to convert your assets into ROM files.
host-templates\html\www\ <-- All your assets will end up here in a binary blob called moaiapp.rom and moaiapp.rom.json.
host-templates\html\www\lib <-- Copy the new moaijs.js file from build\build-html into this folder. You'll have to create the folder first.
host-templates\html\www\ <-- Put this folder onto a server and the index.html should run your game.

Re: How to build HTML5 host?

by halfnelson » Sat Jan 03, 2015 10:40 pm

Daniel. For future reference, the new steps are:

1. bin\build-html.bat (to build libmoai/moaijs.js)
2. util\moaiutil.bat host-html (to create hosts/html folder from template)
3. copy lua src to hosts/html/src
4. run buildrom.bat

I have detailed instructions for the host-templates and new build routines in the docs directory in the repo. See them here: https://github.com/moai/moai-dev/blob/d ... Windows.md

currently working on a helper (like moaicli) for streamlining this ( i have done android and windows already) to be "moaiutil host init" then edit the config.lua then "moaiutil host build html" ( or moaiutil host run android-gradle or moaiutil host run windows)
