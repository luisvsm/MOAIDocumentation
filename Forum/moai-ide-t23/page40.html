Re: Moai IDE?

by paulclinger » Tue Aug 21, 2012 7:20 pm

I'd like to revive this discussion. I've been working on a generic Lua IDE (ZeroBrane Studio: https://github.com/pkulchenko/ZeroBraneStudio) and one of the features it includes is a remote debugger. I'm familiar with Koneki/LDT, Textadept and other IDEs that support Lua, but ZBS includes some of the features that may not present anywhere else (live coding, Markdown integration, static analysis among other things), and makes these and other features available in a compact and extensible package.

The remote debugger is socket based and has been shown to work locally on the same computer with different environments (Lua, LuaJIT, Love2d, moai host), but also across the network with apps running on different machines or mobile devices (tested with mosync). I haven't yet tested it with moai running on a mobile device, but plan to do it shortly. The IDE runs on Windows and Mac with a Linux version being worked on (I have several demos posted here: http://notebook.kulchenko.com/mt/mt-sea ... e&limit=20).

Paul.
