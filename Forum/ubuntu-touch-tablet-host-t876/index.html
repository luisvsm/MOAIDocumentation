Ubuntu Touch/Tablet host ..

by ibisum » Wed Jun 13, 2012 8:37 am

Hallo MOAI'ers,

I've recently built an Ubuntu Tablet development system (PandaboardES + 12.1" multi-touch capacitive Touchscreen + Ubuntu 12.04) and am planning on working on the Linux host support for MOAI so that, on an Ubuntu Tablet, MOAI behaves just like iOS/Android hosts.

I'm of the opinion that Ubuntu Tablet systems should be a real easy, obvious target for MOAI. With such host support, we can roll out our MOAI-based apps into yet another appstore eco-system, and I don't really think there's anything at all negative about participating in the emerging Ubuntu Tablet scene as soon as possible ..

Is anyone else interested in this direction of things, and if so - can we collaborate/coordinate our work efforts? spacepluks' recent efforts to get the Linux host build caught up with other hosts has been a tremendous boost to this - but up next is a bit of work to take advantage of the Ubuntu tablet features, multi-touch, etc. Unfortunately, I can't seem to get a real commitment from the core MOAI dev team to roll Linux host build support into the next SDK - perhaps if we let them know the interest in this, they'll put a bit of a higher priority on rolling spacepluks' changes in with the next SDK release?
