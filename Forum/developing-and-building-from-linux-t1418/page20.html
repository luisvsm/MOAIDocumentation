by ibisum » Mon Dec 03, 2012 10:32 am

For CURL make sure you have the "curl-*-dev" packages for your system installed - and then delete your build/ directory and re-make/configure once you've confirmed you've got files in /usr/include/curl.. this will then turn on the curl feature - if you don't have the -dev package, auto-detection of libcurl won't happen and your MOAI client won't have it built-in. Note that there are different 'flavours' of libcurl in Ubuntu - you have to choose one, and install its -dev packages with it. I'm using libcurl-openssl-dev on my Ubuntu 12.04/amd64 machine, and it works fine. EDIT: also, its more like:

Code: Select all
  1.  

  2. local c = require "curl"

  3.  



.. I think.

As for the click events on Ubuntu, thats' very interesting - would you have some demo/sample snippet that replicates the behaviour? I'm not seeing the issue myself .. I've just finished building spacepluks' new host though, and can help test/debug if you provide a snippet.
