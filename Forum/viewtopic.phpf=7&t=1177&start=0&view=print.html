I'm trying to run the app-tapjoy sample, but I believe I haven't compiled in all the necessary lua-modules for it to work properly from my host.

Posted: Thu Sep 06, 2012 8:07 pm

So I figured this one out. You don't include the lua modules into the host per se. The problem is that samples/config/config.lua is pointing to the wrong location. Replace the contents in config.lua with the following path will solve the problem:
Code: Select all
  1. package.path = package.path .. ';../../../src/lua-modules/moai/?.lua'

Hope this will be corrected in the public repo.
