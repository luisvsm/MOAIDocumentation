Ant host project errors

Posted: Wed Aug 22, 2012 7:41 pm
by lancew
When I run run-host.sh with cygwin, it creates all the correct projects as expected, but when you import those projects into Eclipse, it has tons of this error in many files across all projects:

The method methodName() of type new Type(){} must override a superclass method

Removing all of the @Override directives fixes the errors and the project builds/compiles successfully. This wouldn't really be a big deal, except that you have to do this every time you need to re-run the shell script.

EDIT: I'm using eclipse version 3.7.2, windows 7 64bit, and cygwin 1.7.x
