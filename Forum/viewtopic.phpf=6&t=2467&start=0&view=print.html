Untz (which we will likely be making mods to, or replacing with something based on PortAudio for the forthcoming Linux release)

We added a bunch of features to moai, most of which are detailed here

The designer used texture packer and GLEED (I don't think physicseditor was used - will update this after I speak with our gameplay programmer)

We're looking at spriter and spine for the next game, but we've got a lot of technical debt to pay off (hence me making noises about merging upstream and getting some sanity back into moai-dev ) before we start the next Moai project. And yes - assuming the game sells, there will be another Moai project
