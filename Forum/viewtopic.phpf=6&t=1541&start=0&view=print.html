--------------
How to install:

1. Close ZeroBrane Studio.
2. Copy the rapanuiapi.lua in ...\ZeroBraneStudio\api\lua\
3. Copy the rapanui.lua in ...\ZeroBraneStudio\interpreters
4. Start ZeroBrane Studio and go to the menu Project --> Lua Interpeter --> Rapanui
