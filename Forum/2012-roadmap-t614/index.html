Re: 2012 Roadmap

by robin.baumgarten » Tue Mar 27, 2012 12:53 pm

Looks good!

Great to see that the documentation will get some more love!
In the long run, I'd really like to see one or two more fleshed-out examples, involving multiple screens, showing how to organize objects and data structures in lua. Having only used lua for tiny scripts before myself, it's been a bit of a hunting-gathering experience to get the workflow right for a full-fledged game.

I totally agree with the emphasis on 2D and isometry. As you mentioned, other platforms such as Unity have more tools for 3D built in, which I think would unnecessarily bloat moai. I like how it is a lean and fast platform now.

I'd imagine the cloud features will take a while for developers to pick up, as we've got to develop the games first. That said, I'm planning to use real-time multiplayer on my game, and I'm going to use the cloud features for that. Looking forward to see how that works!
