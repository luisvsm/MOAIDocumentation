by Nodepond » Thu Jul 31, 2014 2:31 pm

Ok, we finally managed it - there is no other way, than doing it the Android hard way. This tutorial helped a lot:
http://ankitthakkar90.blogspot.de/2013/ ... -with.html

Because of some oddities in our project with compressed textures, we could not use a normal .obb asset-storage. But what we did, was some sort of hack of loading the original MOAI-generated .apk as .obb and mounted it into the MoaiActivity:

Code: Select all
  1. String filePath = Helpers.getSaveFilePath(this);

  2. String fileName = Helpers.getExpansionAPKFileName(this, true, num);

  3.                        

  4. Moai.mount ( "bundle", filePath + "/" + fileName );

  5. Moai.setWorkingDirectory ( "bundle/assets/lua" );
