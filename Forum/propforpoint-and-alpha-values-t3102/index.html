Code: Select all
  1.  

  2.

  4. if  pick:getAttr(MOAIColor.ATTR_A_COL) < 1 then

  5. --recursively call propForPoint

  6. end

  8.

Re: propForPoint and Alpha values

by Vavius » Tue Mar 31, 2015 11:48 pm

If you're using MOAI 1.6 then there is something built-in for this. You can set hit mask (image) to MOAIDeck and use HIT_TEST_FINE to test against pixels in the mask.

See "samples/gfxQuad2D-inside".
