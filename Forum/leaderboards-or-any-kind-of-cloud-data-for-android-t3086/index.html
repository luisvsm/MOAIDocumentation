by orkj » Fri Mar 13, 2015 3:51 pm

Sure. I am using it for one of our leaderboards (the best players in multiplayer).

I am using the MOAIGooglePlayServicesAndroid class (https://moaiforge.github.io/moai-sdk/ap ... droid.html) available on 1.5. Initial work was done way back in the day, but I finished it it up and sent in a PR which is in 1.5. So I am not really that knowledgeable about it, since the majority of the work was done by someone else.

Basically what you want to do, is make sure the app_id property gets set in your strings.xml (I think that was related to google play services). Oh, and of course also include google-play-services in settings-global.sh

Other than that, just follow set up tutorials for a regular java project (making the leaderboard and such).

Good luck!
