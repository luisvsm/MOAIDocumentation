Go to your environment variables (Right click on My Computer -> Settings -> Advanced system settings -> Environment Variables).
In the System Variables list, find PATH. Add the path to your Cygwin binary folder at the very beginning, so that it gets searched before Windows itself.

This should fix all the "File not found - .?*" errors, and probably leading to everything else to work too.
