This is the code causing it:

Code: Select all
  1.         #ifndef AKU_WITH_TEST

  2.                 glutIgnoreKeyRepeat ( 1 );

  3.  

  4.                 glutKeyboardFunc ( _onKeyDown );

  5.                 glutKeyboardUpFunc ( _onKeyUp );

  6.                 glutSpecialFunc ( _onSpecialFunc );

  7.  

  8.                 glutMouseFunc ( _onMouseButton );

  9.                 glutMotionFunc ( _onMouseDrag );

  10.  

  11.                 #ifdef FREEGLUT_STATIC

  12.                 glutMultiButtonFunc ( _onMultiButton );

  13.                 glutMultiMotionFunc ( _onMultiMotion );

  14.                 #endif

  15.                

  16.                 glutPassiveMotionFunc ( _onMouseMove );

  17.     #endif



in GlutHost.cpp.

Removing all the #ifdef lines makes inputs work again, so what you said is the problem, thanks!

Update: from halfnelson, changing #ifndef AKU_WITH_TEST to #if !AKU_WITH_TEST should fix it as well.
