Hi,

 

I'm new to Moai (hence the post in the New User's forum).

 

I set up a demo project according to the MoaiProjectSetup guide. Now I'm trying to run my app and lua can't find any files that I require in.

 

Here's my structure:

host_windows -> holds moai.exe and windows host files

lua -> holds main.lua and campaign.lua

run.bat -> ﻿.host_windowsmoai.exe ".luaconfig.lua" ".luamain.lua"

 

In main.lua I require a file campaign.lua.

local c = require("campaign") and get the following errors:

Looks like it's searching in host_windows and host_windows\lua for your Lua files. I'm looking at the Wolf Toss project now. We have it set up (something) like this:

WolfToss\run.bat

WolfToss\host-windows\moai.exe

WolfToss\lua\main.lua

The run.bat first makes 'lua' the working directory before running then uses a relative path to moai.exe:

pushd lua

"..\host-windows\moai" "main.lua"

popd

Hope that helps!

Another solution I found was to set the LUA_PATH as follows:

  

set LUA_PATH = ?;?.lua;\path\to\game\?.lua;\path\to\game\?.dll
... run moai ...
