Android build Beta 9 on windows

by 519n1ff16 » Sun Nov 20, 2011 4:38 pm

The R.java error is a typical error that happens when you import an Android project into Eclipse. Clean/refresh the project in Eclipse a few times until it generates R.java, and the error should go away. Alternatively, go build the android project yourself with Ant, which should also generate R.java.



The @Override error is because your compiler compliance level is set to 1.5, not 1.6. Go to preferences>java>compiler in Eclipse and change the compiler compliance level to 1.6.
