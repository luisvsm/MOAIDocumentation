Reduce texture size on loading?

Posted: Sat Dec 15, 2012 1:25 pm
by Fotomanni
Hello,

is it possible to reduce the size of a texture while loading it or later in memory? Without a change to the host.

E.g. I use PNG files with 1920x1080 pixels. This ist the maximum screen resolution which I use. But if the user decide to use a lower resolution it would be nice to load the textures in the lower resolution. Of course I can supply files in different resolutions but this will blow up the size of the distribution heavily.

Regards,
Manfred

Re: Reduce texture size on loading?

Posted: Sun Dec 16, 2012 9:58 am
by Muzz
Yep, perfectly possible.

You load textures in when you call them. Pretty sure you just need to swap the prop to a new deck with the smaller/larger texture in it.
That being said, including downsized files wont blow out your filesize as much as you may think, seeing as every half resolution is a quarter filesize.
