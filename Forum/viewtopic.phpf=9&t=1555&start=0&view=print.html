Nothing wrong with multiple viewports! The viewport is not a real object per se, it is just a definition of how things should scale on the screen. Like for one layer the "scalesettings" (viewport) has one setup, on another layer there is another setup. A viewport wont steal a part of screenspace or anything like that. The layer will ask the viewport: "How should I scale these sprites when drawing on screen?" and viewport will answer. It doesn't actually draw the sprites.
