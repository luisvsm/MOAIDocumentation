Splash screen required

Alternatively, a note on the credits screen.

Rob0 wrote:
Compiles to each format through what programs

With native tools for each platform:
  • Android sdk & ndk toolchain
  • Xcode for iOS. Requires Mac OS X
  • Visual Studio for windows 8 mobile

You should install appropriate tools to build the app. MOAI is not a "one button to build all" solution like some other closed source SDKs. Sometimes you'll have to dig into native code.

MOAI is open source, there is no hidden analytics, ads, enforced splash screens. Any native glitch/problem can be fixed with some effort. Any new native integration can be added. Learning curve is steep, but it definitely pays off later.

Rob0 wrote:
Is your simulator also the compiler? Is the compiled code indecipherable by common means?

There are different options, it's all up to you. You can leave your plain text Lua scripts in the app bundle, or compile them into lua bytecode. I prefer using squish (it makes single file from all lua sources) and embedding it in C++ header which is compiled into the binary.
