I didn't check your code yet but if you're having trouble with alpha transparency not working then you should try adding the MOAIImage.PREMULTIPLY_ALPHA flag when you're loading an image. Like so:
Code: Select all
  1. img = MOAIImage.new()

  2. img:load( 'img.png', MOAIImage.PREMULTIPLY_ALPHA )


The flag also works with MOAIImageTexture in case you're using that instead of MOAIImage.
