Free ReSharper and RubyMine licenses

Posted: Thu Aug 06, 2015 8:42 am
by DanielWolf
Great news for everybody who uses Visual Studio: We just acquired 30 open source licenses for ReSharper Ultimate.

In case you don't know it: ReSharper is a Visual Studio plugin that makes navigating, writing, and refactoring source code much easier. We use it at work for C# and I couldn't imagine going back. Now the latest versions also support C++. Be sure to check it out!

Also, we have 4 open source licenses for RubyMine. So if you want to work on moaicli or another Ruby-based project related to Moai, this might be for you.

If you're interested in a license, just send me a private message and I'll give you a link.

These are one-year licenses specifically for open source purposes. That means that although you could do so, you are not allowed to use them for anything but work on the Moai SDK.
