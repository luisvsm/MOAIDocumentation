Lua is an embedded scripting language, that means by itself, it doesn't do much. A host program will provide it functionalities. Moai Cloud is one such host that gives it db and web access.
The piece of code you posted is meant for a Moai host.
SciTE is a text editor bundled with a very basic lua host, that's why it can't execute your code.
You can still use it to edit though, and you may even configure it to run moai instead of lua interpreter.

I suggest reading this wiki article first:
/wiki/index.php?title ... top_Basics

About installation: You can extract Moai SDK wherever you want to. You can refer to this article for more details: /wiki/index.php?title ... ject_Setup
But basically, extract it where you feel convenient.
