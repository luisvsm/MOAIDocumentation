Since Moai is open source, I don't feel bad about asking people to pitch in.

I had some thoughts on Rob's request for Glyph Designer support. Really looks like a fantastic product. Anyone interested in writing an importer for Glyph Designer and/or Angelcode? MOAIFont supports rendering from a sprite sheet with kerning tables. Setup is under the hood (right now it's all generated automatically from TTF), but if someone wants to help out I'd be happy to collaborate by exposing the internal glyph initialization API.

I've written some JSFL examples that show how you can grab sprite sheets, flipbook animations and tweened animations our of Flash, but there are probably some other (less expensive tools) people would like to use. Same deal: if anyone wants to volunteer, I'm ready to advise or support you.
