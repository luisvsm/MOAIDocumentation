When you run, you need to make sure the path in the batch file.

*run.bat
Code: Select all
  1.  

  2. cd /d %~dp0

  3.  

  4. :: setting

  5. set MOAI_HOME=D:\Program\moai-sdk  <------ Here!!!

  6. set MOAI_BIN=%MOAI_HOME%\bin\win32\moai.exe

  7. set DEST_DIR=dest

  8. set CONFIG_LUA=%MOAI_HOME%\samples\config\config.lua

  9. set MAIN_LAU=main.lua

  10.  

  11. :: build

  12. call ant build  <------ Need Apatch ant

  13.  

  14. :: run

  15. cd %DEST_DIR%

  16. %MOAI_BIN% %CONFIG_LUA% %MAIN_LAU%

  17.  

  18.  



*run.sh
Code: Select all
  1.  

  2.  

  3. #!/bin/bash

  4. cd `dirname $0`

  5.  

  6. # setting

  7. MOAI_HOME=~/moai-sdk    # Here!

  8. DEST_DIR=./dest

  9. CONFIG_LUA="${MOAI_HOME}/samples/config/config.lua"

  10. MAIN_LAU="main.lua"

  11.  

  12. # build

  13. ant build     # Need Apatch ant!

  14. STATUS=$?

  15.  

  16. # run

  17. if [ $STATUS == "0" ]; then

  18.     cd ${DEST_DIR}

  19.     ${MOAI_HOME}/bin/osx/moai $CONFIG_LUA $MAIN_LAU

  20. fi

by makotok1123 » Fri Mar 29, 2013 3:12 pm

The origin of the objects used in the Flower is (0, 0).
Changed by the function "setPiv" center point.

Code: Select all
  1.  

  2. local layer = flower.Layer()

  3.  

  4. local image1 = flower.Image("cathead.png")

  5. image1:setLayer(layer)

  6. image1:setPiv(0, 0, 0)

  7. image1:setLoc(50, 50, 0)

  8.  

  9. local image2 = flower.Image("cathead.png")

  10. image2:setLayer(layer)

  11. image2:setPiv(10, 10, 0)

  12. image2:setLoc(50, 50, 0)

  13.  

  14. local image3 = flower.Image("cathead.png")

  15. image3:setLayer(layer)

  16. image3:setPivToCenter()

  17. image3:setLoc(50, 50, 0)

  18.  



Without directly modifying the Flower, you can change the behavior of the code from the outside.
Because it's a dynamic language Lua.

If the position of the image is (0, 0), and you want to display in the center of the screen?

Code: Select all
  1.  

  2. module(..., package.seeall)

  3.  

  4. local viewport = MOAIViewport.new()

  5. viewport:setSize(flower.screenWidth, flower.screenHeight)

  6. viewport:setScale(flower.viewWidth, -flower.viewHeight)

  7.  

  8. function onCreate(e)

  9.     layer = flower.Layer()

  10.     layer:setViewport(viewport)

  11.     layer:setScene(scene)

  12.  

  13.     image1 = flower.Image("cathead.png")

  14.     image1:setLoc(0, 0, 0)

  15.     image1:setLayer(layer)

  16. end
