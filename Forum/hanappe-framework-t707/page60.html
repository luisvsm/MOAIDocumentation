Depends on the configuration of the Pivot.

/docs/class_m_o_a_i_prop.html
/docs/class_m_o_a_i_transform.html

Code: Select all
  1.  

  2. module(..., package.seeall)

  3.  

  4. local viewport = MOAIViewport.new()

  5. viewport:setSize(flower.screenWidth, flower.screenHeight)

  6. viewport:setScale(flower.viewWidth, -flower.viewHeight)

  7.  

  8. function onCreate(e)

  9.     layer = flower.Layer()

  10.     layer:setViewport(viewport)

  11.     layer:setScene(scene)

  12.    

  13.     group = flower.Group()

  14.     group:setLayer(layer)

  15.     group:setSize(200, 200)

  16.     group:setPiv(100, 100, 0) -- group:setPivToCenter()

  17.     group:setLoc(0, 0, 0)

  18.  

  19.     image1 = flower.Image("cathead.png", 100, 100)

  20.     image1:setPos(0, 0)

  21.  

  22.     image2 = flower.Image("cathead.png", 100, 100)

  23.     image2:setPos(100, 100)

  24.  

  25.     group:addChild(image1)

  26.     group:addChild(image2)

  27.    

  28. end

  29.
