I found, what I think is, a bug unless this is the way that it is expected to work.

Code: Select all
  1. module(..., package.seeall)

  2.  

  3. function onCreate(params)

  4.     layer = Layer()

  5.     layer:setScene(scene)

  6.  

  7.     g1 = Graphics({left = 0, top = 0,width = 100, height = 50, layer = layer})

  8.     g1:setPenColor(1, 0, 0, 1):fillCircle()

  9.     g1:setPenColor(0, 1, 0, 1):drawRect()

  10.     g1:setLeft(50)

  11.     g1:setTop(50)

  12. end



The code above should produce a red circle inside a green box.
However due to the way Moai or OpenGL works, the centre of the circle is at the corner of the square. To fix this, I have changed two functions (drawCircle and fillCircle are effected) in hp/display/Graphics.lua:
Code: Select all
  1. function M:drawCircle(x, y, r, steps)

  2.     x = x or 0 -- if x does not exist, use 0

  3.     y = y or 0

  4.     x = (self:getWidth() + x)/2 --find centre of shape

  5.     y = (self:getHeight() + y)/2

  6.     steps = steps and steps or 360

  7.     local command = function(self)

  8.         if x and y and r and steps then

  9.             MOAIDraw.drawCircle(x, y, r/2, steps) --r is halved to fit inside the Graphics bounds.

  10.         else

  11.             local rw = math.min(self:getWidth(), self:getHeight())

  12.             MOAIDraw.drawCircle(x, y, rw/2, steps)

  13.         end

  14.     end

  15.     table.insert(self.commands, command)

  16.     return self

  17. end

  18.  

  19.  

  20.  

  21. function M:fillCircle(x, y, r, steps)

  22.     x = x or 0

  23.     y = y or 0

  24.     x = (self:getWidth() + x)/2

  25.     y = (self:getHeight() + y)/2

  26.     steps = steps and steps or 360

  27.     local command = function(self)

  28.         if x and y and r and steps then

  29.             MOAIDraw.fillCircle(x, y, r/2, steps)

  30.         else

  31.             local r = math.min(self:getWidth(), self:getHeight())

  32.             MOAIDraw.fillCircle(x, y, r/2, steps)

  33.         end

  34.     end

  35.     table.insert(self.commands, command)

  36.     return self

  37. end

Do you mean that you want to remove the MOAIProp from MOAILayer?

Code: Select all
  1.  

  2. function onCreate(params)

  3.     layer = Layer()

  4.     layer:setScene(scene)

  5.    

  6.     -- Remove sprite.

  7.     sprite = Sprite {texture = "moai.png", layer = layer}

  8.     sprite:setLayer(nil)

  9.     -- layer:removeProp(sprite) -- Case 2

  10.  

  11.     -- Remove group

  12.     group = Group {layer = layer}

  13.     group:addChild(sprite)

  14.     group:setLayer(nil)

  15.     -- group:removeChild(sprite) -- If you want to remove the Sprite

  16.  

  17. end
