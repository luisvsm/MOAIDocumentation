Re: RSS Link Question

by naturally » Wed Jan 28, 2015 2:56 am

I think this is it:

Code: Select all
  1. Options +FollowSymlinks

  2. RewriteEngine On

  3. RewriteBase /

  4. RewriteRule ^rss$ rss.xml [NC,L]



Seems to work!
