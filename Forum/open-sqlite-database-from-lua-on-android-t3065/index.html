I found this page /forums/post5301.html?hilit=sqlite%20android#p5301 which seemed like it would be very helpful, but I found that MOAIEnvironment.documentDirectory is nil so that example dies at line 9.

Can anyone share a code snippet/example showing opening an SQLite file from Lua? Here is what I have working on Windows (borrowed from some example):

Code: Select all
  1.  

  2. require "luasql"

  3. local sqlite = luasql.sqlite3()

  4. local DB = sqlite:connect("DB_NAME.sqlite")

  5.  

  6. --== Basics

  7. assert( DB:execute("create table if not exists XYZ( field1 varchar(10), field2 smallint )" ))

  8. assert( DB:execute("insert into XYZ (field1, field2) values( 'foo', 3)") )

  9.  

  10. -- Dump a table to output

  11. local result = DB:execute("select * from XYZ")

  12. local row = {}

  13. while result:fetch(row) do

  14.     print( table.concat(row, '|') )

  15. end

  16. result:close()

  17. result = nil

  18.  

  19. -- == Clean up

  20. DB:close()

  21. sqlite:close()
