You must pause your 'actions' (animations, box2dworld... etc) 'manually' via yourobject:pause(true)
when you want to resume actions then you can call yourobject:pause(false)
I have a Gamestate for Pause to draw Pause on screen and wait for touch/keys to resume game

Re: How to Pause and Resume Game ?

by poetowen » Mon Jan 07, 2013 7:30 pm

Hi KaKaVip,
I received your email, and here is the code I used.
Code: Select all
  1. -- Pause

  2. oldRoot = MOAIActionMgr.getRoot ()

  3. MOAIActionMgr.setRoot ()

  4.  

  5. local thread = MOAIThread.new ()

  6. thread:run ( doPaused )

  7.  

  8. -- doPaused

  9. function doPaused ()

  10.   MOAIInputMgr.device.touch:setCallback( onTouchEvent )

  11. end

  12.  

  13. -- Resume

  14. MOAIActionMgr.setRoot ( oldRoot )


Hope it helps.
