Thanks Motig!

- Stunt Zone started nicely, but we're not quite happy with the DAU numbers at the moment. Anyway it's a learning process, we've been trying all kinds of things to see what works and what doesn't. MemoReaction project was one of those learning projects as well to see how this genre compares with more advanced games and to get some comparison data.

We're planning to make a one more bigger(ish) update for Stunt Zone and then decide if we're gonna completely drop the project or put more efforts on it. The overall engine is good, maybe release another more arcade kind of game on top of it later and concentrate on user retention.

Also playing with idea of making a bigger multiplayer game later, but this would require some external funding.. Any VC's online?
