@ignisDeveloper,

Thinking about this even more if you are just trying to keep object with higher or lower y's I wrote up the following. It shouldn't matter if they touch

Code: Select all
  1.  

  2. screenHeight = 768  --make this the max y value of your levels

  3. HighestOnTop = true -- make false and they will be the other order depending on your needs

  4. for i, v in ipairs(myGroup:getAllChildren()) do

  5.         if HighestOnTop then

  6.            v:setLevel(v.y)

  7.         else

  8.            v:setLevel(screenHeight-v.y)

  9.         end

  10.     end

Hi Sergeyka,

it is not simple to understand what you need.
Maybe you want to create five aliens and each one should be 60 pixel away from the other.

Try to use this:

Code: Select all
  1. local left = 60

  2. for i = 1, 5 do

  3.     RNFactory.createImage("images/alien" .. i .. ".png", { top = 30, left = 10 + left * i })

  4. end



or maybe this, if you need somehow to use tables:

Code: Select all
  1. local left = 60

  2. local assets = { "alien1", "alien2", "alien3", "alien4", "alien5" }

  3. for i = 1, 5 do

  4.     RNFactory.createImage("images/" .. assets[i] .. ".png", { top = 30, left = 10 + left * i })

  5. end

by mattfortunati » Mon Oct 01, 2012 8:17 pm

Hi Gianmichele,
yes, RapaNui can handle multitouch.
This:
Code: Select all
  1. function touch(event)

  2. for i,v in pairs(event) do print(i,v) end

  3. end

  4.  

  5. RNListeners:addEventListener("touch",touch)


should output something like this:
y 265
x 122
name
id -1
phase began
tapCount 0

the id is the touch id, finger id (from 0 to max and "-1" in Moai simulator) while tapCount is the "double tap" count of the current id.

by mattfortunati » Sat Oct 06, 2012 1:53 am

Hi btatarov,

you can insert animations into groups after their creation.
Code: Select all
  1. local group=RNGroup:new()

  2. local aChar = RNFactory.createAnim(self.image, { sizex = charWidth, sizey = charHeight } )

  3. group:insert(aChar)



The auto-add into groups is just an old and deprecated way to do this.
