by mattfortunati » Thu Jan 12, 2012 4:15 am

Hi Jeremy, i checked out your code and I made it work.



First of all your timer (the one in splash.lua) should pass "1"  as third argument instead of "0" to call the "leave" function just one time.

RNMainThread.addTimedAction(2000, leave, 1, nil)



Then, you were right , the director doesn't work well if we hide a scene and then we show another one.

But I solved your problem like this:



1) in test.lua we add both our scenes in the director:



director:addScene("rapanui-samples/games/test/splash")

director:addScene("rapanui-samples/games/test/game")



2)in splash.lua instead of hiding a scene with "leave" function and showing a new one in the onEnd we directly show the second scene in the "leave" function, and the director will do the fade job between scenes for us.

This is how I changed the "leave" function:



local function leave()

  director:showScene("rapanui-samples/games/test/game","fade")

end



3)so in splash.lua we don't need any additional code for the onEnd function leaving it just like in samples scenes.



If you make this simple changes, your code should work properly. (It does for me).



Im glad you like our work, let me know ;D
