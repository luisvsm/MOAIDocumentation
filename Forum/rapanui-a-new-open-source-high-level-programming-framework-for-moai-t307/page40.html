by mattfortunati » Wed Feb 15, 2012 2:12 pm

Hi Sergeyka,

Code: Select all
  1. function Cock

  2.  temp = math.random(480)

  3.  temp = math.random(480)

  4.  cock_anim1.x = temp

  5. end




The above code will cause an error because the function lacks parenthesis.
You should declare functions like this:

Code: Select all
  1. function functionName()

  2.     --do something

  3. end



So the right code for you will be :
Code: Select all
  1. function Cock()

  2.  temp = math.random(480)

  3.  temp = math.random(480)

  4.  cock_anim1.x = temp

  5. end
