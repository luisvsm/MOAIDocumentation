by stefano.linguerri » Tue Sep 11, 2012 9:37 pm

Hi mnewnam,

This code should fit your needs:

Code: Select all
  1.  

  2. text = RNFactory.createText("Touch on buttons", { size = 18, top = 200, left = 5, width = 250, height = 50 })

  3.  

  4. back = RNFactory.createImage("images/background-purple.png")

  5. back:sendToBottom()

  6.  

  7. buttonOne = RNFactory.createImage("images/tile1.png", { top = 50, left = 50 })

  8. buttonOne.name = "Button1"

  9.  

  10. buttonTwo = RNFactory.createImage("images/tile2.png", { top = 50, left = 150 })

  11. buttonTwo.name = "Button2"

  12.  

  13. buttonTwo.rotation = 30

  14.  

  15.  

  16.  

  17. function buttonTouchDown(event)

  18.     text:setText("Button " .. event.target.name .. " touch down!")

  19. end

  20.  

  21. function buttonMoved(event)

  22.     text:setText("Button " .. event.target.name .. " moved!")

  23. end

  24.  

  25. function buttonUP(event)

  26.     text:setText("Button " .. event.target.name .. " touch up")

  27. end

  28.  

  29. buttonOne:setOnTouchDown(buttonTouchDown)

  30. buttonOne:setOnTouchMove(buttonMoved)

  31. buttonOne:setOnTouchUp(buttonUP)

  32.  

  33. buttonTwo:setOnTouchDown(buttonTouchDown)

  34. buttonTwo:setOnTouchMove(buttonMoved)

  35. buttonTwo:setOnTouchUp(buttonUP)

  36.  

  37.  



let me know if it's ok, I'll add this example to rapanuidev branch.

Re: RapaNui: a new, open-source, high-level programming fram

by BlueByLiquid » Wed Sep 12, 2012 2:22 am

@ignisdeveloper,

Each frame you could check the y value and call myObjectInGroup:setLevel() according to their y value.

for example:

Code: Select all
  1. if doObjectsIntersect(o1, o2)  then

  2.    if o1.y > o2.y then

  3.       o1:setLevel(o2:getLevel() + 1)

  4.    else

  5.       o2:setLevel(o1:getLevel() + 1)

  6.    end

  7. end

  8.  



I don't know if RapaNui has an a function "doObjectsIntersect()" but shouldn't be too difficult to write that and will be dependents on your images/shapes anyway.
