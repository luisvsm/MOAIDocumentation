You can build Windows x86 apps with Moai, and there are several under development now. As Romain noted, Moai is based on OpenGL so you'll need to create a robust Windows host that supports OpenGL.

We don't currently offer support any Windows 8 specific features, or the new variants such as Windows 8 RT or Windows 8 Phone. We haven't seen a lot of demand from our community for those platforms.
