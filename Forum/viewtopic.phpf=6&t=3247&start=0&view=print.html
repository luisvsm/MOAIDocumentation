Hey everyone, I just want to share a bit about what I'm doing because it is too quiet here!


I've been working on an action adventure game since early last year. It's targeting Steam ( and possibly some console platform in future )

A key feature of the project is the visual style we're using. We are combining pixel-art with 3D lighting to create a 'modern' yet 'retro' graphics style. The lighting is based on Deferred Shading technology. Although it's not more than a very common 3D technology, thanks to seamless collaboration with the artist, we made something surprisingly appealing.






A little more about the tech side:

I did some modification to MOAI's render pipeline to support deferred shading. It was actually an easy job thanks to the clean structure of Patrick's great work.

Of course I have to mention my IDE. It's a little beast being built along with my previous projects. Although I've decided to make it open-source, I've been busy working on the game project since then. Hopefully I'm gonna get back to clean up the messy and release it some time this year.


What I wanted to say is we can use MOAI to do a lot more things than we thought.
It's awesome!

PS: If you're interested in my project, you can follow us on the twitter ( @pixpilgames ) or tumblr ( http://pixpilgames.tumblr.com ).

Cheers!
