Re: Ability to play videos?

by thebullno1 » Tue May 15, 2012 2:42 am

chrisjohnson wrote:I know this is kind of old, but I was hoping there have been some updates recently. We really need some way to play video in the app for cut scenes, intros and things like that. What do you recommend? I'm not a C programmer, but I do know Lua, Corona, and HTML5. Let me know what can be done in Moai.

As far as I know (yesterday), there's still no support for video. You can try working around with HTML5 with web view.
There's no web view for Android right now. In iOS, you can try pointing the webview to a HTML file that plays the video onload. However, you still have little control over it.

Moai is written using C/C++ so it's not easy to extend it without touching C/C++.

An alternative choice is to try scripting. If you scenes are not that complex, you can script it and MOAICoroutine should make the process easier.
