by mikegriffin » Fri Aug 31, 2012 1:25 pm

I have incorporated a similar idea into
https://github.com/latestpost/moaigamekit2.0

You can use the following in main.lua to pull down code.

In the gamekit above I have a menu button that pulls down the code on demand. Even if the particular module downloaded fails or is corrupted the front end menu can still be run the code, the button pushed and the game code can be pulled down again.

This has been tested on Windows, IOS and Android


Code: Select all
  1.  

  2. function getFile(filename,url)

  3.  

  4. directory=MOAIEnvironment.documentDirectory

  5. if MOAIEnvironment.osBrand=="Windows" then directory=nil end

  6. if directory~=nil then

  7.         filename=directory.."/"..filename

  8. end

  9. if directory==nil then

  10.         directory=""

  11.         filename=filename

  12. end

  13.  

  14. gfilename=filename

  15.  

  16. --print ("getting file"..filename)

  17. capture=assert(curl.new());

  18.  

  19. local file=assert(io.open(filename, "wb"));

  20.  

  21. assert(capture:setopt(curl.OPT_WRITEFUNCTION, function (stream, buffer)

  22. stream:write(buffer)

  23. return string.len(buffer);

  24. end));

  25.  

  26. assert(capture:setopt(curl.OPT_WRITEDATA, file));

  27. assert(capture:setopt(curl.OPT_PROGRESSFUNCTION, function (_, dltotal, dlnow, uptotal, upnow)

  28. end));

  29.  

  30. assert(capture:setopt(curl.OPT_NOPROGRESS, false));

  31. assert(capture:setopt(curl.OPT_BUFFERSIZE, 5000));

  32. --assert(c:setopt(curl.OPT_HTTPHEADER, "Connection: Keep-Alive", "Accept-Language: en-us"));

  33. assert(capture:setopt(curl.OPT_URL, url));

  34. assert(capture:setopt(curl.OPT_CONNECTTIMEOUT, 45));

  35. --assert(c:perform());

  36.  

  37. capture:perform({writefunction = function(str)

  38. end})

  39.  

  40. assert(capture:close()); -- not necessary, as will be garbage collected soon

  41. s="abcd$%^&*()";

  42. assert(s == assert(curl.unescape(curl.escape(s))));

  43. file:close();

  44.  

  45. end

  46.  

  47. filename="game_download.lua"

  48. getFile(filename,"http://www.innovationtech.co.uk/moai/gamekit/game_download.lua")

  49.                                

  50.  

  51. directory=MOAIEnvironment.documentDirectory

  52. if MOAIEnvironment.osBrand=="Windows" then directory=nil end

  53. if directory~=nil then

  54.         filename=directory.."/"..filename

  55. end

  56. if directory==nil then

  57.         directory=""

  58.         filename=filename

  59. end

  60. dofile(filename)
