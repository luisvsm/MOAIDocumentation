samples/ios/app-tapjoy

The first one is old, and has not been maintained. It uses a tapjoy.lua Lua module to hit Tapjoy's web API. We have switched over to using Tapjoy's native SDK which is what the second sample provides an example of.

When you set up virtual currency on your Tapjoy dashboard, you are asked to provide a callback URL if you are maintaining the virtual currency yourself. This URL gets hit every time a user completes an offer. It lets you know which user and how much currency to award. We do not have a sample on how to do that, however, Tapjoy's knowledge center would be a good place to start: http://knowledge.tapjoy.com/.
